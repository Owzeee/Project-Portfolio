Description of all the functionalities available:


1. Clients are able to do the following:
 Manage (create, edit, and delete) an account with required information (name, address…),
 Associate a license/serial number (that have been purchased from a third party) with an
account with required information (serial number, purchase date, expiry date…),
 List serial numbers of a client with all required information (serial number, software name, purchase date, expiry date…),
 Acquire new licenses,
 Renew (or stop renewing) licenses (for subscription-based),

2. Software providers are able to:
 Manage clients’ account,
 Generate serial numbers,
 Associate serial numbers with products and clients,
 Enable/unblock and disable/block a serial number (if needed),
